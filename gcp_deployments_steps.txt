This document assumes you already have your local git repo available and pointed to your remote repo.

Basic order of commands for end-to-end deployment.

LOCAL REPO
Edit, save, stage, commit and push to remote repo. In our case github.com
>> git add .
>> git git commit -m "your commit message"
>> git push

GCP CLOUD SHELL
Open gcp cloud shell and clone the github repo to cloud shell local storage
>> git clone https://github.com/your-github-repo-url.git
